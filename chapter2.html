Number System

A number system is a writing system for denoting numbers logically using digits or other symbols. It provides a unique representation of every number and reflects the arithmetic and algebraic structure of numbers. In computing, several number systems are utilized:


Decimal (Base-10): The standard system for denoting integer and non-integer numbers, using digits from 0 to 9.

Binary (Base-2): Utilized internally by almost all modern computers and computer-based devices because it is straightforward to implement with digital electronic circuitry. It uses only two digits: 0 and 1.

Octal (Base-8): Uses digits from 0 to 7. Each octal digit represents three binary digits, making it more compact for representing binary numbers.

Hexadecimal (Base-16): Uses digits from 0 to 9 and letters A to F. Each hexadecimal digit represents four binary digits, providing a more human-friendly representation of binary-coded values.
Understanding these systems is crucial for tasks such as data representation, memory addressing, and low-level programming.

Boolean Algebra: Boolean algebra is a branch of mathematics that deals with variables that have two distinct values: true and false, typically denoted as 1 and 0, respectively. It forms the foundation of digital logic and is essential for designing and understanding computer circuits and algorithms. Key concepts include: Logical Operations: AND, OR, and NOT operations that define how binary variables interact. Truth Tables: Tables that list all possible values of logical variables and the result of applying logical operations to them. Simplification of Expressions: Techniques like De Morgan's laws and Karnaugh maps are used to simplify complex Boolean expressions, optimizing digital circuits. Proficiency in Boolean algebra is vital for tasks such as circuit design, algorithm optimization, and understanding the underlying principles of computer operations.

1. Binary to Decimal Conversion
To convert a binary number (base 2) to decimal (base 10), you multiply each digit by the power of 2 and sum the results.

Steps:

Write the binary number. Starting from the right, assign each digit a power of 2 (2^0, 2^1, 2^2, etc.). Multiply each binary digit by the corresponding power of 2. Sum the results. Example: Convert 1011 (binary) to decimal.

Start from the right:

1 = 2^3 = 8
0 = 2^2 = 0
1 = 2^1 = 2
1 = 2^0 = 1
Now sum the results:
8 + 0 + 2 + 1 = 11 (decimal)
So, 1011 (binary) = 11 (decimal).

2. Decimal to Binary Conversion

To convert a decimal number (base 10) to binary (base 2), divide the number by 2, recording the remainder each time. Then, read the remainders from bottom to top.

Steps:
Divide the decimal number by 2. Write down the remainder. Continue dividing by 2 until the quotient is 0. The binary number is the string of remainders read from bottom to top.

3. Binary to Octal Conversion
To convert binary to octal (base 8), group the binary digits in sets of three, starting from the right. If there are not enough digits to make a group of three, add leading zeros. Then, convert each group of three digits to its octal equivalent.

Steps: Group the binary number into sets of three digits, starting from the right. Convert each group to octal.

Example: Convert 110101 (binary) to octal.

Group the digits:
110 101
Convert each group to octal:
110 (binary) = 6 (octal)
101 (binary) = 5 (octal)
So, 110101 (binary) = 65 (octal).

4. Octal to Binary Conversion

To convert octal to binary, convert each octal digit to its 3-bit binary equivalent.

Steps:
Write each octal digit in its 3-bit binary equivalent.

Example: Convert 35 (octal) to binary.

Convert each octal digit to binary:
3 (octal) = 011 (binary)
5 (octal) = 101 (binary)
So, 35 (octal) = 011101 (binary).

5. Binary to Hexadecimal Conversion

To convert binary to hexadecimal (base 16), group the binary digits in sets of four, starting from the right. If there are not enough digits to make a group of four, add leading zeros. Then, convert each group of four binary digits to its hexadecimal equivalent.

Steps:
Group the binary number into sets of four digits, starting from the right. Convert each group to hexadecimal.

Example: Convert 1011101 (binary) to hexadecimal.

Group the digits:
1011 101
Add leading zeros to make a full group of four:
1011 1101
Convert each group to hexadecimal:
1011 (binary) = B (hexadecimal)
1101 (binary) = D (hexadecimal)
So, 1011101 (binary) = BD (hexadecimal).

6. Hexadecimal to Binary Conversion

To convert hexadecimal to binary, convert each hexadecimal digit to its 4-bit binary equivalent.

Steps:
Write each hexadecimal digit in its 4-bit binary equivalent.

Example: Convert A3 (hexadecimal) to binary.

Convert each hexadecimal digit to binary:
A (hexadecimal) = 1010 (binary)
3 (hexadecimal) = 0011 (binary)
So, A3 (hexadecimal) = 10100011 (binary).


Truth Tables


And Gate:


The AND gate outputs 1 only if both inputs are 1. If either or both inputs are 0, the output will be 0


OR Gate :


The OR gate outputs 1 if either or both of the inputs are 1. It outputs 0 only when both inputs are 0


NAND Gate:


The NAND gate is the opposite (negation) of the AND gate. It outputs 1 unless both inputs are 1. When both inputs are 1, the output is 0


NOR Gate :


The NOR gate is the opposite (negation) of the OR gate. It outputs 1 only when both inputs are 0. If either or both inputs are 1, the output is 0


XNOR Gate:


The XNOR (Exclusive NOR) gate is the negation (NOT) of the XOR gate. It outputs 1 when both inputs are the same (either 0,0 or 1,1), and it outputs 0 when the inputs are different


NOT Gate :


The NOT gate, also known as an inverter, is one of the simplest logical gates in digital electronics. It is a unary operator, meaning it operates on a single input, and its function is to invert or negate the input value.


XOR Gate:


The XOR (Exclusive OR) gate outputs 1 when exactly one of the inputs is 1. If both inputs are the same (either 0,0 or 1,1), the output is 0

Binary Subtraction

Binary subtraction is a key operation in digital electronics, and there are several methods to perform it. The most commonly used techniques involve 1's complement and 2's complement, which allow subtraction to be performed by adding binary numbers.

1s' compliment

1's complement is a binary representation system used to represent negative numbers. It works by flipping all the bits of a given binary number—changing 0s to 1s and 1s to 0s. This system is one of the early methods used in digital computers to handle signed numbers (both positive and negative).


Steps to Perform 1's Complement:
Write the binary number: Start with the binary number you want to find the 1's complement of. Invert the bits: Flip every bit in the binary number:

Change every 0 to 1.
Change every 1 to 0.
The result is the 1's complement: After inverting all the bits, you get the 1's complement of the number.
Example 1: 1's Complement of a Positive Number

Let’s say we want to find the 1's complement of the binary number 1010.

Step 1: Start with 1010.
Step 2: Invert the bits:
1 becomes 0
0 becomes 1
1 becomes 0
0 becomes 1
Step 3: The result is 0101.
So, the 1's complement of 1010 is 0101.

A = 9 (binary representation: 1001)
B = 7 (binary representation: 0111)
Step 2: Find the 1's complement of the number to subtract
Find the 1's complement of B, which is the number you're subtracting. To do this, invert all the bits of B.
B = 7 (binary: 0111)
Invert the bits: 1000 (1's complement of 7)
Step 3: Add the 1's complement of B to A
Now, add the 1's complement of B (which is 1000) to A (which is 1001), as if it were a normal binary addition.

Binary Subtraction Using 2's Complement Binary subtraction can be simplified by using 2's complement because it allows us to perform subtraction as an addition operation. By converting the number to be subtracted into its 2's complement form (which represents the negative of the number), subtraction becomes the same as adding that negative number.
Steps for Binary Subtraction Using 2's Complement:
Let's walk through the steps of subtracting two binary numbers using 2's complement.

Step-by-Step Process:
Write the binary numbers you want to subtract.
Find the 2's complement of the number you're subtracting (this gives the negative value).
Add the 2's complement of the number you're subtracting to the first number.
Handle the carry:
If there's a carry out (i.e., a leftmost carry bit), discard it.
The result will be the correct difference.


Boolean Algebra - Grade 11 Questions & Answers

1. What is Boolean Algebra?

Boolean Algebra is a branch of algebra that deals with binary variables and logic operations. It is used in digital logic design and computer science for circuit simplifications and logical expressions.

It is based on three basic operations:

  • AND (⋅): Outputs true if both inputs are true.
  • OR (+): Outputs true if at least one input is true.
  • NOT (¬): Inverts the input value.

Boolean Algebra follows a set of rules and theorems to simplify logical expressions.

2. What are the basic laws of Boolean Algebra?

Boolean Algebra follows several fundamental laws:

  • Commutative Law: A + B = B + A and A ⋅ B = B ⋅ A
  • Associative Law: (A + B) + C = A + (B + C) and (A ⋅ B) ⋅ C = A ⋅ (B ⋅ C)
  • Distributive Law: A ⋅ (B + C) = (A ⋅ B) + (A ⋅ C)
  • Identity Law: A + 0 = A and A ⋅ 1 = A
  • Complement Law: A + A' = 1 and A ⋅ A' = 0
  • Idempotent Law: A + A = A and A ⋅ A = A

These laws are used to simplify Boolean expressions for logic circuit design.

3. What are De Morgan’s Theorems?

De Morgan’s Theorems are important rules used to simplify Boolean expressions:

  • First Theorem: ¬(A ⋅ B) = ¬A + ¬B
  • Second Theorem: ¬(A + B) = ¬A ⋅ ¬B

These theorems help in converting AND operations to OR operations and vice versa, simplifying logic circuit implementation.

4. Explain the use of Karnaugh Map (K-Map) in Boolean Algebra.

Karnaugh Map (K-Map) is a visual method used to simplify Boolean expressions. It organizes truth table values into a grid where adjacent cells differ by a single variable.

Steps to simplify using K-Map:

  • Draw a K-Map based on the number of variables (2, 3, or 4 variables).
  • Fill in the K-Map with truth table values.
  • Group adjacent 1s in powers of 2 (1, 2, 4, 8, etc.).
  • Write the simplified Boolean expression.

K-Maps reduce complex expressions, making circuit design more efficient.

5. What is a logic gate? Explain different types of logic gates.

A logic gate is an electronic circuit that performs Boolean operations. The basic types of logic gates include:

  • AND Gate: Outputs 1 if both inputs are 1.
  • OR Gate: Outputs 1 if at least one input is 1.
  • NOT Gate: Inverts the input (1 becomes 0, and 0 becomes 1).
  • NAND Gate: Combination of AND and NOT (inverted AND output).
  • NOR Gate: Combination of OR and NOT (inverted OR output).
  • XOR Gate: Outputs 1 if inputs are different.
  • XNOR Gate: Outputs 1 if inputs are the same.

Logic gates are used to build digital circuits in computing and electronics.
