Introduction to C

C is a general-purpose, procedural programming language developed by Dennis Ritchie in 1972 at Bell Labs. It is one of the most widely used programming languages and serves as the foundation for many modern languages like C++, Java, and Python.

C is known for its simplicity, efficiency, and flexibility. It is used for system programming, embedded systems, game development, and more.

Features of C

C has several features that make it a powerful and popular programming language:

  • Procedural Language: Follows a step-by-step procedural approach.
  • Portability: Programs written in C can run on different platforms with minimal changes.
  • Efficiency: Provides low-level access to memory and hardware.
  • Modularity: Supports functions and libraries for code reusability.
  • Extensibility: Allows the addition of new features and libraries.

Structure of a C Program

A C program typically consists of the following components:

                
#include <stdio.h> // Preprocessor directive

int main() { // Main function
    printf("Hello, World!"); // Statement
    return 0; // Return statement
}

Components of a C Program

  • Preprocessor Directives: Includes header files (e.g., #include <stdio.h>).
  • Main Function: Entry point of the program (int main()).
  • Statements: Instructions to perform tasks (e.g., printf("Hello, World!");).
  • Return Statement: Indicates the end of the program (return 0;).

C Preprocessors and Header Files

Preprocessors are directives that instruct the compiler to preprocess the source code before compilation. They begin with a # symbol.

Common Preprocessor Directives

  • #include: Includes header files (e.g., #include <stdio.h>).

Header Files

Header files (e.g., stdio.h) contain function declarations and macros. They are included using the #include directive.

Characters Used in C

C uses the following types of characters:

  • Letters: A-Z, a-z
  • Digits: 0-9
  • Special Characters: ! @ # $ % ^ & * ( ) _ + - = { } [ ] | \ : ; " ' < > , . ? /
  • Whitespace: Spaces, tabs, and newlines.

Comments in C

Comments are used to explain code and make it more readable. They are ignored by the compiler.

Types of Comments

  • Single-Line Comments: Start with //. 
    •                     
// This is a single-line comment
  • Multi-Line Comments: Enclosed in /* ... */. 
    •                     
/* This is a
   multi-line comment */

Identifiers in C

Identifiers are names given to variables, functions, arrays, etc. They must follow these rules:

  • Can contain letters, digits, and underscores (_).
  • Must begin with a letter or underscore.
  • Cannot be a keyword.
  • Are case-sensitive.

Example of Identifiers

                
int age; // "age" is an identifier
float salary; // "salary" is an identifier

Keywords in C

Keywords are reserved words in C that have special meanings. They cannot be used as identifiers.

Common Keywords

                
int, float, char, if, else, for, while, return, void, struct, etc.

Tokens in C

All the things written in code are tokens. Tokens are the smallest individual units in a C program. They include:

  • Keywords: Reserved words (e.g., int, return).
  • Identifiers: Names of variables, functions, etc.
  • Constants: Fixed values (e.g., 10, 3.14).
  • Strings: Sequence of characters (e.g., "Hello").
  • Operators: Symbols for operations (e.g., +, -).
  • Special Symbols: Punctuation marks (e.g., ;, {).

Data Types in C

Data types define the type of data a variable can hold. C supports the following basic data types:

  • int: Integer values (e.g., int age = 25;).
  • float: Floating-point numbers (e.g., float pi = 3.14;).
  • char: Single characters (e.g., char grade = 'A';).
  • double: Double-precision floating-point numbers (e.g., double salary = 45000.50;).
  • void: Represents no value (used in functions and pointers).

Constants and Variables

Variables are used to store data that can change during program execution. Constants are fixed values that cannot be changed.

Variables

                
int age = 25; // Variable declaration and initialization
age = 30;    // Value can be changed

Constants

                
const float PI = 3.14; // Constant declaration
// PI = 3.15; // Error: Cannot change a constant

Type Specifiers

Type specifiers modify the basic data types to provide more control over their size and behavior. Common type specifiers include:

  • short: Reduces the size of int (e.g., short int x;).
  • long: Increases the size of int or double (e.g., long int y;).
  • signed: Allows positive and negative values (default).
  • unsigned: Allows only positive values (e.g., unsigned int z;).

Simple and Compound Statements

Statements are instructions that perform actions in a program.

Simple Statements

A single instruction ending with a semicolon (;).

                
int x = 10; // Simple statement

Compound Statements

A block of statements enclosed in curly braces ({ }). 

                
if (x > 5) { // Compound statement
    printf("x is greater than 5");
    x = x + 1;
}

Arithmetic Operators

Arithmetic operators are used to perform mathematical operations like addition, subtraction, multiplication, division, and modulus.

Common Arithmetic Operators

  • +: Addition
  • -: Subtraction
  • *: Multiplication
  • /: Division
  • %: Modulus (remainder)

Example

                
int a = 10, b = 3;
int sum = a + b; // 13
int difference = a - b; // 7
int product = a * b; // 30
int quotient = a / b; // 3
int remainder = a % b; // 1

Relational Operators

Relational operators are used to compare two values. They return a boolean result (1 for true, 0 for false).

Common Relational Operators

  • ==: Equal to
  • !=: Not equal to
  • >: Greater than
  • <: Less than
  • >=: Greater than or equal to
  • <=: Less than or equal to

Example

                
int a = 10, b = 20;
int result1 = (a == b); // 0 (false)
int result2 = (a < b);  // 1 (true)

Logical Operators

Logical operators are used to combine multiple conditions. They return a boolean result (1 for true, 0 for false).

Common Logical Operators

  • &&: Logical AND
  • ||: Logical OR
  • !: Logical NOT

Example

                
int a = 1, b = 0;
int result1 = (a && b); // 0 (false)
int result2 = (a || b); // 1 (true)
int result3 = !a;       // 0 (false)

Assignment Operators

Assignment operators are used to assign values to variables. They can also perform operations while assigning.

Common Assignment Operators

  • =: Simple assignment
  • +=: Add and assign
  • -=: Subtract and assign
  • *=: Multiply and assign
  • /=: Divide and assign
  • %=: Modulus and assign

Example

                
int a = 10;
a += 5; // a = 15
a -= 3; // a = 12
a *= 2; // a = 24
a /= 4; // a = 6
a %= 5; // a = 1

Unary Operators

Unary operators operate on a single operand. They are used to perform operations like increment, decrement, and negation.

Common Unary Operators

  • ++: Increment
  • --: Decrement
  • -: Negation
  • !: Logical NOT

Example

                
int a = 5;
a++; // a = 6
a--; // a = 5
int b = -a; // b = -5
int c = !a; // c = 0 (false)

Conditional Operator

The conditional operator (? :) is a ternary operator used for decision-making. It evaluates a condition and returns one of two values based on the result.

Syntax

                
condition ? expression1 : expression2;

Example

                
                  int a = 10, b = 20;
                  int max = (a > b) ? a : b;
                  printf("%d is greater\n", max);

I/O Functions

Input and output functions allow interaction with the user. Common functions are printf for output and scanf for input.

Example

            
#include <stdio.h>

int main() {
    int num;
    printf("Enter a number: ");
    scanf("%d", &num);
    printf("You entered: %d", num);
    return 0;
}

if Statement

The if statement executes a block of code if a specified condition is true.

Example

            
int a = 10;
if (a > 5) {
    printf("a is greater than 5\n");
}

if-else Statement

The if-else statement allows for one block of code to execute if the condition is true and another if it is false.

Example

            
int a = 10;
if (a > 15) {
    printf("a is greater than 15\n");
} else {
    printf("a is not greater than 15\n");
}

if-else if-else Statement

This statement allows multiple conditions to be checked sequentially.

Example

            
int a = 10;
if (a > 20) {
    printf("a is greater than 20\n");
} else if (a > 10) {
    printf("a is greater than 10 but not 20\n");
} else {
    printf("a is 10 or less\n");
}

Nested if Statement

A nested if statement allows an if statement if. It works like and operator. inside another

Example

            
int a = 10, b = 20;
if (a < 20) {
    if (b > 15) {
        printf("Both conditions are true\n");
    }
}

Switch Statement

The switch statement allows multiple cases to be checked based on an expression. It also takes a default value.

Example

            
int day = 3;
switch (day) {
    case 1:
        printf("Monday\n");
        break;
    case 2:
        printf("Tuesday\n");
        break;
    case 3:
        printf("Wednesday\n");
        break;
    default:
        printf("Other day\n");
}

Iteration Control Statements

Loops are used to execute a block of code repeatedly.

While Loop

            
int i = 1;
while (i <= 5) {
    printf("%d\n", i);
    i++;
}
1 2 3 4 5

Do-While Loop

            
int i = 1;
do {
    printf("%d\n", i);
    i++;
} while (i <= 5);
1 2 3 4 5

For Loop

            
for (int i = 1; i <= 5; i++) {
    printf("%d\n", i);
}
1 2 3 4 5

Programming in C - Grade 11 Questions & Answers

1. What is C programming?

C is a general-purpose, procedural programming language developed by Dennis Ritchie in 1972. It is widely used for system programming, game development, and embedded systems.

C is known for its efficiency, portability, and use of structured programming techniques.

2. What are the basic features of C?

C has several important features:

  • Simple and Efficient: Provides direct control over hardware.
  • Portable: Code can be compiled on different platforms.
  • Structured Language: Uses functions and loops for better organization.
  • Supports Low-level Manipulation: Allows direct access to memory.
  • Fast Execution: Used in performance-critical applications.

3. What is the basic structure of a C program?

A C program consists of:

  • Preprocessor Directives: Instructions like #include.
  • Main Function: int main() where execution starts.
  • Variables and Data Types: Used to store values.
  • Functions: Blocks of reusable code.
  • Statements and Expressions: Instructions for computation.

Example:

        #include <stdio.h>

        int main() {
            printf("Hello, World!");
            return 0;
        }

4. What are Data Types in C?

Data types define the type of data stored in a variable.

  • int: Integer numbers (e.g., 10, -5).
  • float: Decimal numbers (e.g., 3.14, -0.99).
  • char: Single character (e.g., 'A', '5').
  • double: Larger decimal numbers.
  • void: No value (used for functions).

Example:

        int age = 18;
        float price = 99.99;
        char grade = 'A';

5. What are Operators in C?

Operators perform operations on variables and values.

  • Arithmetic Operators: +, -, *, /, %
  • Relational Operators: ==, !=, <, >, <=, >=
  • Logical Operators: &&, ||, !
  • Bitwise Operators: &, |, ^, ~, <<, >>
  • Assignment Operators: =, +=, -=, *=, /=, %=

Example:

        int a = 5, b = 2;
        int sum = a + b;  // 7
        int isEqual = (a == b);  // 0 (false)

6. What are Conditional Statements in C?

Conditional statements control the flow of a program based on conditions.

  • if: Executes a block if a condition is true.
  • if-else: Executes one block if true, another if false.
  • else-if: Checks multiple conditions.
  • switch-case: Alternative to multiple if-else.

Example:

        int num = 10;
        if (num > 0) {
            printf("Positive");
        } else {
            printf("Negative");
        }

7. What are Loops in C?

Loops repeat a block of code multiple times.

  • for Loop: Runs a specific number of times.
  • while Loop: Runs while a condition is true.
  • do-while Loop: Runs at least once before checking condition.

Example:

        for (int i = 1; i <= 5; i++) {
            printf("%d ", i);
        }

8. What are Functions in C?

Functions are reusable blocks of code that perform specific tasks.

Types:

  • Built-in Functions: printf(), scanf()
  • User-defined Functions: Functions created by the programmer.

Example:

        void greet() {
            printf("Hello, World!");
        }

        int main() {
            greet();
            return 0;
        }

9. What are Arrays in C?

Arrays store multiple values of the same type.

Types:

  • One-dimensional Array: Stores a list of values.
  • Multi-dimensional Array: Stores a matrix-like structure.

Example:

        int numbers[5] = {1, 2, 3, 4, 5};
        printf("%d", numbers[0]); // Output: 1

10. What is Pointer in C?

Grade 12

A pointer is a variable that stores the memory address of another variable.

Example:

        int x = 10;
        int *ptr = &x;
        printf("%d", *ptr); // Output: 10

11. What is File Handling in C?

Grade 12

File handling allows reading and writing data to files.

Example:

        FILE *fptr;
        fptr = fopen("test.txt", "w");
        fprintf(fptr, "Hello, C!");
        fclose(fptr);
